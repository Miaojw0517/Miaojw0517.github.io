vantui中不支持v-model

uniapp 中的 van-field 组件是基于 Vant Weapp 组件库的，并不是原生的 Vue 组件，因此它不支持 v-model 指令。v-model 是 Vue 中用于双向数据绑定的语法糖，它等价于 value 和 input 事件的结合。
为了解决这个问题，你需要手动绑定 value 和 input 事件来实现数据的双向绑定。
